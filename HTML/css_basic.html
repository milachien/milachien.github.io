Think Cafe No.5

Expresso

Think Cafe No5 使用的是維也納咖啡品牌-Julius Meinl的豆子，1862年起家，現在已經傳承到第五代了。Espresso喝起來溫醇順口，不會太濃郁，口味偏酸、帶些甘甜。去年咖啡展時也有不少國外品牌的咖啡豆商參展，尤其是義大利、來的還不少。有的參展廠商還只許店家試喝、不開放給一般民眾試喝呢。想必國外廠商是看多台灣咖啡市場的發展才會來試試水溫。
